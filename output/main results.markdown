
## Results Data

Due to file size limitations, all geospatial results are hosted externally.

**Download Link:**  
https://drive.google.com/drive/folders/1ejAbZoMRbcP63sV5El0Yjr2aGaV5xgb-?usp=drive_link

**Available Raster Files:**

| File Type | Description | Method |
|-----------|-------------|---------|
| `ES for mites.tif` | Published environmental suitability for mites | BRT |
| `ES for scrub typhus.tif` | Predicted environmental suitability for scrub typhus | RF |
| `10fold_sb_sd.tif` | Prediction uncertainty | 10-fold spatial block validation |
| `100fold_bootstrap_sd.tif` | Model uncertainty | 100-fold clustered bootstrap |
| `MOP_strict_extrapolation.tif` | Strict extrapolation zones | Mobility-Oriented Parity (MOP) |
